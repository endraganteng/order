# order

Project aplikasi order berbasis Laravel.

## Menjalankan project

1. Salin environment:
   - `cp .env.example .env`
2. Install dependency PHP:
   - `composer install`
3. Install dependency frontend:
   - `npm install`
4. Jalankan aplikasi:
   - `php artisan serve`
5. Jalankan build frontend:
   - `npm run build`
